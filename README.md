# grommunio Desk

The native desktop application for [grommunio](https://www.grommunio.com) running on Windows, Mac and Linux.

Developed using [Electron](https://www.electronjs.org)

## Usage

### Installation

[Download](https://grommunio.com/download/) the desktop app and install it.

### Configuration
- `~/.config/grommunio-desk/config.json` on Linux

### Logs
- `~/.config/grommunio-desk/logs/main.log` on Linux
